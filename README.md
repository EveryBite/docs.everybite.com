# EveryBite Developer Documentation

Developer documentation for the EveryBite platform, hosted at [docs.everybite.com](https://docs.everybite.com).

## Products

- **Menu API** - Access menu data, search, filter, and calculate nutrition
- **Passport** - Universal dining identity with preferences, saved dishes, and loyalty
- **Ordering API** - Cart, checkout, and fulfillment (separate product)

## Development

Preview locally:

```bash
npm i -g mintlify
mintlify dev
```

## Deployment

This documentation is automatically deployed via [Mintlify](https://mintlify.com) when changes are pushed to the `main` branch.
