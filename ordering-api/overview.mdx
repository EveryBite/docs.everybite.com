# Ordering API

<Info>
The Ordering API is a separate product. Contact sales@everybite.com for access.
</Info>

The Ordering API enables cart management, checkout, and order fulfillment for EveryBite-enabled restaurants.

## Integration with Menu API

The Ordering API works seamlessly with the Menu API:

```graphql
# 1. Query dishes with Menu API
query {
  dishes(menuKey: "...", filters: {...}) {
    results { dish { id, name, price } }
  }
}

# 2. Add to cart with Ordering API
mutation {
  addToCart(dishId: "dish_123", quantity: 1) {
    cart { items { ... } }
  }
}
```

## Integration with Passport

When a diner has an EveryBite Passport:

- **Saved addresses** auto-populate for delivery
- **Loyalty programs** apply automatically
- **Meal history** records the order
- **Preferences** ensure allergen warnings display
